Url Shortener Microservice

The microservice is able to make the link shorter. It returns JSON data.

How does a response look?

https://url-shorten-ms.glitch.me/new/https://www.google.com

https://url-shorten-ms.glitch.me/new/https://foo.com

JSON output will be:

{ "original_url":"https://google.com", "short_url":"https://url-shorten-ms.glitch.me/111" }

Created link redirect to original source

https://url-shorten-ms.glitch.me/111
