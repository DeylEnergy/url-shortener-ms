Url Shortener Microservice

The microservice is able to make the link shorter. It returns JSON data.

How does a response look?

https://little-url.herokuapp.com/new/https://www.google.com

https://little-url.herokuapp.com/new/https://foo.com

JSON output will be:

{ "original_url":"https://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }

Created link redirect to original source

https://little-url.herokuapp.com/8170
